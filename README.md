# Bottega Diner

Creating a Vanilla Javascript Diner Menu which allows for selection of items, choosing a price fixed menu or regular items, receiving a total dinner bill and responses from a waitress.

![Diner Menu](https://dl.airtable.com/.attachmentThumbnails/6aba4e72488b1b9af0e9f9f082b2ac5a/ecbe0aeb)

So far using the Math package of the Lodash library.
